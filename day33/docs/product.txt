# FlowCast — платформа для управления подкастами

## Что такое FlowCast

FlowCast — облачная платформа для независимых подкастеров и небольших студий.
Позволяет записывать, редактировать, публиковать и монетизировать подкасты
из одного интерфейса без установки дополнительного ПО.

## Основные возможности

### Запись и редактирование
- Запись через браузер (Web Audio API) или загрузка готовых файлов (MP3, WAV, FLAC, M4A)
- Встроенный редактор: обрезка, склейка, удаление пауз, шумоподавление
- Автоматическая нормализация громкости по стандарту LUFS -16
- Поддержка многодорожечной записи до 4 участников одновременно (план Studio и выше)

### Публикация
- Автоматическая генерация RSS-фида, совместимого с Apple Podcasts, Spotify, Google Podcasts, Яндекс.Музыка
- Кастомный домен для подкаста (например, podcast.yourdomain.com)
- Планировщик публикаций: настройте дату и время выхода эпизода заранее
- Автоматическая транскрипция эпизодов (поддерживаются русский и английский языки)

### Аналитика
- Статистика прослушиваний в разбивке по эпизодам, платформам и географии
- Retention-график: видно, на какой минуте слушатели отключаются
- Уникальные слушатели, динамика роста аудитории, топ эпизодов
- Данные обновляются раз в 6 часов

### Монетизация
- Платные подписки для слушателей (Stripe, ЮKassa)
- Динамическое размещение рекламы (DAI) — вставка рекламных роликов без перезаписи эпизода
- Донаты прямо со страницы подкаста

## Технические требования

- Браузер: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- Для записи: микрофон, стабильное интернет-соединение от 5 Мбит/с
- Мобильная запись: приложение FlowCast для iOS 15+ и Android 10+
- Максимальный размер одного файла при загрузке: 2 ГБ
- Поддерживаемые форматы экспорта: MP3 (128/192/320 кбит/с), WAV, AAC

## Безопасность и данные

- Все файлы хранятся в зашифрованном виде (AES-256)
- Передача данных только по HTTPS/TLS 1.3
- Резервное копирование ваших файлов выполняется ежедневно
- GDPR-совместимость: вы можете запросить экспорт или удаление всех данных
- Двухфакторная аутентификация доступна для всех планов
